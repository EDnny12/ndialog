# NDialog 
Custom dialog with blur background and popup animation, Made with love by [Moch. Nizwar Syafuan](https://fb.com/nizwar.richardo) ‚ù§

## Let's Code!

``` dart
showDialog(
  context: context,
  builder: (context) {
    return NDialog(
      blur: 5,
      content: Text("Hi, This is the content of dialog that you click."),
      title: Text("NDialog"),
      actions: <Widget>[
        FlatButton(
          padding: EdgeInsets.only(),
          child: Text("You"),
          onPressed: () {},
        ),
        FlatButton(
          padding: EdgeInsets.only(),
          child: Text("Are"),
          onPressed: () {},
        ),
        FlatButton(
          padding: EdgeInsets.only(),
          child: Text("Awesome"),
          onPressed: () {},
        ),
      ],
    );
  }
);
```
